Reflection

Two main web development topics I learned that I have used in this very site are CSS Grid and jQuery.

Having learned Bootstrap first, it was what I was most comfortable with. However, I felt like after learning CSS grid, it was the better implementation for myself as a web developer. The layout makes more sense to me, and the syntax makes me feel like I have more control over the layout of the page while writing fewer lines of code. It was a challenge for me to implement it on a big scale for the first time with this project, but once it was set up, it made me easier to code the rest of the site than it would have been with Bootstrap. I still have much to learn, but I feel like I have a grasp of the basics.

With jQuery, I only found one way to implement it into my site by toggling the screenshots of each project. Although the implementation is simple, sometimes the most simple approach helps achieve the goal best. I could have used more jQuery throughout the site, but I feel the overall layout is easier to navigate and simpler to read for the user than if I would have inundated the site with jQuery to show off. My full array of jQuery skills are shown through the projects, and not necessarily in this site directly.
